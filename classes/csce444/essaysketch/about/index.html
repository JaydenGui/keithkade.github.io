Essaysketch - The Web isn't as it seems

As the web becomes more and more dynamic, thinking about it in traditional terms is insufficient. The goal of this site is to show that to the user.

Two Roads Diverge

User is presented with a navigational choice between two paths reminiscent of the Robert Frost poem.

Fountain

Juxtaposed with the natural setting, the user finds a urinal labelled as a fountain. It has been places there by some other internet user.

Chasm

The user finds a chasm in the ground. One side is labeled and the other is knowledge. Hopefully this causes the user to question their own conceptual model.

Memex

User generated content serves as a way for web citizens to create their own trails

Crossroads

User sees a sign marking a crossroads. Sign has labels for the directional options.

Portals

User sees portals through which they can travel. Can be thought of as hypermedia links.

Monolith - Landmark / Filter

User sees monolith akin to the one from 2001 a space odyssey. At first there are multiple navigation choices but they fade away as the monolith directs the user.
